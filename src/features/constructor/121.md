# Wedding Invites App — Техническое задание

## Описание

Telegram Mini App для создания и отправки свадебных приглашений.

## Модель монетизации

Freemium + платные тарифы через ЮKassa.

## Срок запуска

7 дней (MVP).

## Структура документации

* `01-architecture.md` — Архитектура проекта
* `02-database.md` — Схема базы данных (Prisma)
* `03-api.md` — API endpoints
* `04-telegram-bot.md` — Flow и код бота
* `05-frontend-components.md` — React компоненты
* `06-payments.md` — Интеграция ЮKassa
* `07-admin.md` — Админка
* `08-roadmap.md` — Roadmap по дням
* `09-env.md` — Переменные окружения

## Как использовать

1. Прочитать все файлы
2. Начать с `02-database.md` и `03-api.md`
3. Параллельно делать `05-frontend-components.md`
4. Интегрировать в конце
